## How to install
1. get the files
2. cd in the dir where the composer.json file is
3. composer update
4. read official documentation on [how to install laravel](http://laravel.com/docs/5.0/installation).
5. migrate db and seed the db
   php artisan migrate
   php artisan db:seed
